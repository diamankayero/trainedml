# trainedml.viz

Ce dossier contient les modules de visualisation avancée du package principal.

## Architecture du dossier

```
viz/
├── bivariate.py
├── boxplot.py
├── correlation.py
├── distribution.py
├── heatmap.py
├── histogram.py
├── line.py
├── missing.py
├── multicollinearity.py
├── normality.py
├── outliers.py
├── profiling.py
├── target.py
├── vizs.py
└── __init__.py
```

Chaque fichier correspond à un type de visualisation ou d'analyse spécifique.
